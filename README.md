# Steam Games analysis
Analysis of Steam games Database

Dataset taken from https://www.kaggle.com/nikdavis/steam-store-games 

The scrapping was taken from official market website of Steam (https://store.steampowered.com/)

The following data was used for analysis: rank, earnings of companies, number of words in names

Main libraries that used: 
  1. Pandas
  2. Seaborn
